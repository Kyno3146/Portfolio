Base de données

Description

Dans ce projet, j'ai créé une base de données pour gérer les stocks et données d'une chocolaterie dont les exigences nous étaient données dans le sujet. J'ai don créé le modèle logique de données (MLD) à partir duquel j'ai généré le script sql pour créer la base de données. J'y ai inséré des données test puis j'ai fait un jeu de tests pour m'assurer de la qualité des relations de ma base de données.
